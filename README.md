# TryHackMe Write-ups

This repository contains my personal write-ups and notes from completing TryHackMe rooms. It serves as a record of my learning progress and a portfolio showcasing the practical skills I've developed in cybersecurity.

## About

The purpose of this repository is to:

- Document my methodology while solving security challenges.
- Reinforce concepts learned through hands-on practice.
- Track my progress as I develop penetration testing and defensive security skills.
- Build a public portfolio of my cybersecurity journey.

## Repository Structure

Each room write-up may include:

- Room overview
- Objectives
- Enumeration process
- Exploitation (where appropriate)
- Privilege escalation (if applicable)
- Tools and commands used
- Key lessons learned

## Skills Covered

Some of the topics covered include:

- Linux
- Windows
- Networking
- Web Security
- Enumeration
- Privilege Escalation
- Nmap
- Gobuster
- Burp Suite
- Metasploit
- Wireshark

## Disclaimer

These write-ups are intended for educational purposes only. They document my personal learning experience and are based on legally accessible TryHackMe labs. I avoid publishing information that would violate TryHackMe's content policies or compromise active challenges.
